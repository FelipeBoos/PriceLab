# Kubernetes - PriceLab

Este diretório contém os manifests Kubernetes do projeto PriceLab.

## Ambiente escolhido

Para desenvolvimento local, o ambiente escolhido foi o Kubernetes do Docker Desktop.

## Pré-requisitos

- Docker Desktop instalado
- Kubernetes habilitado no Docker Desktop
- kubectl instalado

## Validar cluster

```bash
kubectl cluster-info
kubectl get nodes
